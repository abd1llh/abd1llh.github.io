# Bugs

## Mobile / viewport
- Terminal input can get covered by the keyboard on iPhone.
- Mobile Safari can zoom inputs when font size is too small.
- `100vh` can be unstable on mobile browsers.

## Terminal
- Prompt and input row can stack incorrectly.
- Welcome text can feel misaligned on narrow screens.
- Output area needs better scrolling behavior.

## Layout
- Some elements can overflow on smaller screens.
- The dashboard table needs horizontal containment on mobile.

## Language
- Arabic and English content need consistent translation coverage.
- RTL should be applied globally when Arabic is active.
